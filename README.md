# Geolocalizacion de Hospitales de Buenos Aires con Node.js 🌎

## To run this code🚀
`npm install`

#### Add your API KEY from Google at:
📁src <br>
  📁Views<br>
    📑google-map.ejs<br>
      ` <script defer
      src="https://maps.googleapis.com/maps/api/js?key={YOURAPIKEY}&callback=initMap">
      </script>`
